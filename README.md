4. Build tool (such as ninja, GNU Make)

## Build
First, You should to clone this repo.

Then use your tool to compile this.

And, It used lots of C++ 17 feature...

### Powershell

``` Powershell
git clone https://github.com/fzzkill/manyfile.git --depth 1
cd manyfile
cmake build .
cmake --build .
```
